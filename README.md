# Модифицированный таймер для звонков на Node.js

Простое консольное приложение-таймер, которое принимает время и выводит сообщение по истечении указанного времени.

## 🎯 Задание

Модифицировать приложение [таймер](https://github.com/PavelKochetkov/dz-nodejs-timer), чтобы он по окончанию заданного времени не просто выводил, что таймер завершён, но и высылал в систему уведомление со звуком.

**Требование к формату ввода**: время вводится в формате: `1h 5m 10s` или `1 5 10` (1 час, 5 минут, 10 секунд).

## 🚀 Установка и запуск

1. Клонируйте репозиторий.
2. Установите зависимости.
3. Запустите таймер с указанием времени:
   ```bash
   node index.js "1h 5m 10s"
   ```
### Примеры использования текущей реализации

```bash
# 1 час, 5 минут, 10 секунд
node index.js 1h 5m 10s

# Только 30 секунд (0 часов, 0 минут, 30 секунд)
node index.js 0h 0m 30s

# 2 часа и 15 минут (0 секунд)
node index.js 2h 15m 0s

# 5 минут
node index.js 0h 5m 0s
```
### Поведение программы

1. Программа парсит переданную строку времени.
2. Конвертирует время в миллисекунды.
3. Устанавливает таймер на указанный период.
4. По истечении времени выводит сообщение в консоль и показывает уведомление со звуком.
5. Программа завершает работу.

## ⚠️ Обработка ошибок

- Если переданы некорректные числовые значения, выводится сообщение: `Invalid time format`.
- Если не переданы все три аргумента, программа завершится с ошибкой: `Invalid time format`.
- Поддерживаются только целые положительные числа.

## 🔧 Реализация

1. **Чтение аргументов**: Получение трех аргументов из `process.argv`.
2. **Извлечение чисел**: Преобразование строк в числа.
3. **Валидация**: Проверка, что все значения являются положительными числами и все значения переданы.
4. **Конвертация**: Перевод времени в миллисекунды по формуле:
   ```
   миллисекунды = (часы × 3600 + минуты × 60 + секунды) × 1000
   ```
5. **Таймер**: Использование `setTimeout()` для отсчета времени.
6. **Уведомление**: Вывод сообщения `Таймер завершил работу` и уведомления со звуком по завершении.
